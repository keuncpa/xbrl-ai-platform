# DART 재무제표 조회 및 분석 스킬

## 개요

DART(전자공시시스템) Open API를 활용하여 한국 상장기업의 실제 재무제표 데이터를 조회하고, 재무비율 산출 및 기업 간 비교 분석까지 수행하는 종합 재무분석 스킬이다.

## 전체 프로세스

```
[기업명/종목코드 입력]
    ↓
[STEP 1] DART API로 재무제표 조회
    ↓
[STEP 2] 재무상태표/손익계산서 파싱
    ↓
[STEP 3] 재무비율 자동 산출
    ↓
[STEP 4] 분석 리포트/대시보드 생성
    ↓
[엑셀 또는 HTML 결과물 출력]
```

## 사전 준비

### DART API 키 확인
- 사용자에게 DART Open API 키가 있는지 확인한다.
- 키가 없으면 https://opendart.fss.or.kr 에서 발급받도록 안내한다.
- 프로젝트 디렉토리의 `xbrl_platform/` 폴더에 기존 코드가 있으면 활용한다.

### 주요 기업 코드 (자주 조회되는 기업)

| SK하이닉스 | 00164779 | 000660 |
| LG전자 | 00401731 | 066570 |
| 현대자동차 | 00164742 | 005380 |

## STEP 1: DART API 재무제표 조회

### API 엔드포인트
```
GET https://opendart.fss.or.kr/api/fnlttSinglAcntAll.json
```

### 필수 파라미터
| 파라미터 | 설명 | 값 |
|---------|------|-----|
| crtfc_key | API 인증키 | 사용자 제공 |
| corp_code | 기업 고유번호 | 8자리 |
| bsns_year | 사업연도 | 예: 2024 |
| reprt_code | 보고서 코드 | 11011=사업보고서, 11012=반기, 11013=1분기, 11014=3분기 |
| fs_div | 재무제표 구분 | CFS=연결, OFS=별도 |

### Python 코드 패턴
```python
import requests

def fetch_financial_statements(corp_code, year, api_key, fs_div='CFS'):
    url = 'https://opendart.fss.or.kr/api/fnlttSinglAcntAll.json'
    params = {
        'crtfc_key': api_key,
        'corp_code': corp_code,
        'bsns_year': year,
        'reprt_code': '11011',
        'fs_div': fs_div,
    }
    r = requests.get(url, params=params)
    data = r.json()
    if data.get('status') != '000':
        return []
    return data.get('list', [])
```

## STEP 2: 데이터 파싱

DART API 응답에서 재무상태표(`sj_nm == '재무상태표'`)와 손익계산서(`sj_nm in ['손익계산서', '포괄손익계산서']`)를 분리한다.

### 금액 필드
- `thstrm_amount`: 당기 금액
- `frmtrm_amount`: 전기 금액
- `bfefrmtrm_amount`: 전전기 금액

### 계층 구조 판단
```python
LEVEL_0 = ['자산총계', '부채총계', '자본총계', '부채와자본총계']
LEVEL_1 = ['유동자산', '비유동자산', '유동부채', '비유동부채', '지배기업 소유주지분', '비지배지분']
# 나머지는 LEVEL_2 (세부 항목)
```

## STEP 3: 재무비율 산출

### 안정성 지표
- **부채비율** = 부채총계 / 자본총계 × 100
- **유동비율** = 유동자산 / 유동부채 × 100
- **자기자본비율** = 자본총계 / 자산총계 × 100

### 수익성 지표
- **매출총이익률** = (매출액 - 매출원가) / 매출액 × 100
- **영업이익률** = 영업이익 / 매출액 × 100
- **순이익률** = 당기순이익 / 매출액 × 100
- **ROA** = 당기순이익 / 자산총계 × 100
- **ROE** = 당기순이익 / 자본총계 × 100

### 활동성 지표
- **총자산회전율** = 매출액 / 자산총계
- **재고자산회전율** = 매출원가 / 재고자산
- **매출채권회전율** = 매출액 / 매출채권

### 기업 간 비교 분석
- 비교 대상 기업 재무비율 나란히 표시
- 최고/최저 하이라이트

### 출력 형식
- **빠른 조회**: 대화 내에서 텍스트로 요약
- **상세 분석**: 엑셀 파일(.xlsx) 생성 → xlsx 스킬 활용
- **대시보드**: HTML 파일 생성 → 브라우저에서 확인

## 주의사항

- DART API는 분당 요청 수 제한이 있으므로, 다수 기업 조회 시 적절한 딜레이(1초)를 둔다.
- 금액은 원(KRW) 단위로 반환되므로, 가독성을 위해 억원 단위로 표시한다.
- 연결재무제표(CFS)가 기본이며, 사용자가 별도(OFS)를 요청하면 전환한다.
- API 키가 만료되었거나 잘못된 경우, 명확한 에러 메시지를 전달한다.
